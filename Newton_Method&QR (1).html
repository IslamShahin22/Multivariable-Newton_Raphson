Newton Rapson method for solving system of nonlinear equations

To solve system of nonlinear equations using Newton Raphson method you should write the system in a specific structure , that structure is

$$\underline{F(\vec{x})}=F(x_1,x_2,\ldots,x_n)= \begin{pmatrix} f_1\\ f_2 \\ \vdots \\ f_n \end{pmatrix} = \underline{0}$$

where $F$ is a vector containing all functions equal to zero vector

$W(\vec{x})= \begin{bmatrix} \nabla f_1(\vec{x}) \\ \nabla f_2(\vec{x}) \\ \vdots \\ \nabla f_n(\vec{x}) \end{bmatrix} =\begin{bmatrix} \frac{\partial f_1(\vec{x})}{\partial x_1} \ldots \frac{\partial f_1(\vec{x})}{\partial x_n} \\ \vdots \\ \frac{\partial f_n(\vec{x})}{\partial x_1} \ldots \frac{\partial f_n(\vec{x})}{\partial x_n} \end{bmatrix}$

Where $W$ is a two dimentional matrix it's Rows are the gradient of functions ,$W$ must be a nonsingular matrix

the iterative equation is

$\underline{x_{n+1}}=\underline{x_n} - W^{-1}(\underline{x_n})F(\underline{x_n})$

def Newton_Raphson (F,Jacobian,*x_n):
  '''this function generate newton raphson step'''
  a=np.linalg.inv(np.array(Jacobian(*x_n)))  
  b=np.array(F(*x_n))
  x_next=x_n-np.matmul(a,b)
  return x_next

def Gram_Schmidt_QR(A):
  '''this function factorize matrix into QR '''
  Q=np.zeros(A.shape)
  R=np.zeros(A.shape)

  for col in range(A.shape[1]):

    u=A[:,col]
    

    for i in range(col):
      R[i,col]=np.dot(A[:,col],Q[:,i])
      
      u=u-R[i,col]*Q[:,i]

    Q[:,col]=u/np.linalg.norm(u,2)
    R[col,col]=np.dot(A[:,col],Q[:,col]) 

  return Q,R

Using Newton's Method for solving unconstrained optimisation </center>

Newton method is a second ordered method by using a quadratic approximation of taylor series

$f(X+h)=f(X)+ h^T \nabla f(x) +\frac{1}{2}h \nabla^2 f(x) h^T +O(h^3)$ \ where $\nabla^2 f(x)$ is called th Hessian matrix \ so the problem is to solve \

$Min/Max \large f(\underline{X}) $
if $\nabla^2 f(X)>0 x $ then $f(X)$ a convex function , so only one extreme point is the optimum point Hessian matrix $\nabla^2 f(X)$ is positive definate

Assume that $f(X)$ is continous and at least twice diffrentiable

$ \nabla f(X^*)=\underline{0}$ whe $X^*$ is an extreme point(maximum , minimum , or sadle point) \

the itertive step equation $x_{n+1}=x_n-t\nabla^2 f^{-1}(x_n) \nabla f(x_n)$ \ where $t$ is the line search step size, for simplicity take $t=1$

So here we solve system of nonlinear equations $\nabla f(x_i)=0$
